# Content Creator — storyboard vidéo publicitaire (1 Day Script)

Template anglophone pour structurer le script d'une vidéo publicitaire courte (format "1 Day Script"), pensé en double colonne : script (mots) + indications visuelles (comment le montrer).

## Structure du storyboard
1. **Attention Getter** — capter l'attention dès la 1ère seconde : question, affirmation forte, appel direct au spectateur, humour. Toujours accompagné d'une idée visuelle forte pour ne pas se fondre dans la masse.
2. **Identify the Gap** — nommer clairement le problème que l'offre résout. Plus le "gap" est identifiable, plus ça résonne. Montrer plutôt que raconter (analogie : une pub anti-nuisibles montre les insectes, ne se contente pas d'en parler).
3. **How your offer bridges the gap** — présenter l'offre comme LA solution qui comble ce gap, si possible via une démonstration visuelle, en différenciant de la concurrence.
